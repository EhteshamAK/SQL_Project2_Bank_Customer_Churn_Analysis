# Bank Customer Churn Analysis (SQL Project)

## Overview
This project analyzes a bank customer dataset using SQL to explore customer behavior, financial patterns, and churn trends. The dataset contains information such as customer demographics, credit score, account balance, number of products, and whether the customer exited the bank.

The goal of the project is to apply SQL queries to extract insights that could help a bank better understand its customers and identify potential churn risks.

## Dataset
The dataset includes the following key fields:

- CustomerId
- Surname
- CreditScore
- Geography
- Gender
- Age
- Tenure
- Balance
- NumOfProducts
- HasCrCard
- IsActiveMember
- EstimatedSalary
- Exited

## Key Analysis
The analysis focuses on answering the following questions:

- Customer distribution by geography
- Average credit score by gender
- Total account balance by geography
- Customer distribution by number of products
- Credit card ownership among customers
- Top customers with the highest balances
- Average estimated salary by geography
- Customer age distribution
- Customer churn analysis
